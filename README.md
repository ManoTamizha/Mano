# 🛒 Customer Shopping Behavior Analysis

## 📌 Overview
This project contains a dataset named **customer_shopping_behavior.csv** which includes customer purchase patterns, demographics, and product preferences.

## 📊 Dataset Details
The dataset includes the following columns:
- Customer ID
- Age
- Gender
- Item Purchased
- Category
- Purchase Amount (USD)
- Location
- Size
- Color

## 🎯 Objective
- Analyze customer buying behavior
- Identify popular product categories
- Perform SQL queries for insights
- Practice data analytics skills

## 🛠 Tools Used
- SQL (Oracle / MySQL)
- Excel / CSV
- (Optional) Python / Power BI

## 📈 Sample Use Cases
- Customer segmentation
- Sales trend analysis
- Product demand insights
- Data visualization dashboards

## 🚀 How to Use
1. Download the CSV file
2. Import into your database (Oracle / MySQL)
3. Run SQL queries for analysis
